# Internet Radio From Terminal

- slug: internet-radio-from-terminal
- category: tech
- tags: gnu-linux
- date: 2013-03-24

-------------------------

Assuming you have `mplayer` installed, add alias to `~/.bashrc` (example for Radio SI):

````bash
alias ra-si='mplayer -cache 256 http://mp3.rtvslo.si/rsi'
````
